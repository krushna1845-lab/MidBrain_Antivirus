
# 🛡️ MidBrain Antivirus

MidBrain Antivirus is an educational cybersecurity project developed using Python and PySide6.  
It demonstrates core antivirus concepts including hash-based malware detection, real-time file monitoring, junk file removal, and RAM optimization.

---

## 🚀 Features

- 🔍 Hash-Based Malware Detection
- 🧠 Real-Time File Monitoring
- 🧹 Junk File Cleaner
- ⚡ RAM Optimization
- 🎨 Modern GUI built with PySide6

---

## 🛠️ Tech Stack

- Python
- PySide6
- Watchdog (Real-Time Monitoring)
- psutil (System Monitoring)
- Hashlib (File Hashing)
